# blood-aid-nodejs-api

<p><b>Setup basic nodejs project:</b></p>
    <p>install nodejs</p>
   <p> npm init -y (for setting up of json packages)</p>
   <p>npm i express</p>
    



<p><b>To run the program:</b></p>
   <p>node app.js </p> 
    <p>npm run devstart (run this for automatic restart)</p>
